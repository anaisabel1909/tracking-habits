# Tracking Habits

A web application designed to help users track daily habits, monitor progress, and build consistency over time.

This project is also a hands-on learning experience focused on web development fundamentals.

---

## Features (planned)

- Create and manage habits
- Daily habit tracking (done / not done)
- Habit streak visualization
- Progress history
- Local data persistence using LocalStorage
- Clean and simple user interface

---

## Technologies

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage  

> Planned for future versions:
> - React
> - Firebase or PostgreSQL
> - User authentication
> - Data visualization (charts)

---

## Project Goals

- Practice front-end development concepts
- Improve JavaScript logic and data handling
- Learn how to structure and evolve a real project
- Gradually migrate to modern frameworks and tools
